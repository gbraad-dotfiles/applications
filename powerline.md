# Powerline

### check
```sh
command -v powerline-config >/dev/null 2>&1
```

### dnf-install
```sh
sudo dnf install powerline tmux-powerline vim-powerline
```

### pip-install
```sh
pip install --force-reinstall --user powerline-status
```

### apt-install
```sh
sudo apt update
sudo apt install powerline
```
