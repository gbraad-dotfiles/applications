# Fedora dotfiles

## info

## create
```sh
machine gofedora download
machine gofedora create
```

## exists
```sh
machine gofedora exists
```

## start
```sh
machine gofedora start
```

## stop
```sh
machine gofedora stop
```

## rm remove
```sh
machine gofedora rm
```

## run-machine
```
machine gofedora service
```
