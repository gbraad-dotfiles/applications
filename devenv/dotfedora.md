# Fedora dotfiles (container)

### env
```sh
devenv dotfedora env
```

### sys
```sh
devenv dotfedora start
```

### user
```sh interactive
devenv dotfedora user
```

### root
```sh interactive
devenv dotfedora root
```

### screen
```sh interactive
devenv dotfedora screen
```

### rm remove
```sh
devenv dotfedora rm
```

### alias
```sh interactive
apps devenv/dotfedora
```
