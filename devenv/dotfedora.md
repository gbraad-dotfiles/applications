# Fedora dotfiles (container)

### env
```sh
devenv dotfedora env
```

### sys
```sh
devenv dotfedora start
```

### user
```sh
devenv dotfedora user
```

### root
```sh
devenv dotfedora root
```

### screen
```sh
devenv dotfedora screen
```

### rm remove
```sh
devenv dotfedora rm
```

### alias
```sh
app devenv/dotfedora
```
