# Fedora golang (container)

### env
```sh
devenv gofedora env
```

### sys
```sh
devenv gofedora start
```

### user
```sh
devenv gofedora user
```

### root
```sh
devenv gofedora root
```

### screen
```sh
devenv gofedora screen
```

### rm remove
```sh
devenv gofedora rm
```

### alias
```sh
app devenv/gofedora
```
