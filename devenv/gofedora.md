# Fedora golang (container)

## env
```sh
devenv gofedora env
```

## sys
```sh
devenv gofedora start
```

## user
```sh interactive
devenv gofedora user
```

## root
```sh interactive
devenv gofedora root
```

## screen
```sh interactive
devenv gofedora screen
```

## rm remove
```sh
devenv gofedora rm
```

## alias
```sh interactive
apps devenv/gofedora
```
